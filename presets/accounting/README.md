# accounting Preset

freee accounting デモデータのプリセットです。

まずは `accounting/office-demo` を使うと、会計データ・社員立替精算・証憑アップロード・固定資産管理をまとめて試せます。
会計中心のデータだけを見たい場合は `accounting/quickstart`、業種別のシナリオは各サブディレクトリの `preset.json` を参照してください。
EC モール・自社EC向けのデモは `accounting/ecommerce-marketplace` が適しています。Amazon・楽天・Shopify の手数料差引入金、配送費、返品返金、広告費、梱包資材、棚卸資産評価損まで一通り確認できます。
